# 🛡️ Cybersecurity Quiz App

## 📖 Overview

The Cybersecurity Quiz App is a GUI-based application developed using **Python** and **Tkinter**. It helps users test their cybersecurity knowledge through multiple-choice questions. The application automatically calculates the score, percentage, and grade at the end of the quiz.

---

## ✨ Features

- GUI-based application using Tkinter
- 10 Multiple Choice Questions
- Randomized question order
- Question progress display (Question X of 10)
- Automatic score calculation
- Percentage calculation
- Grade evaluation (A+, A, B, C, Fail)
- Interactive and user-friendly interface

---
